DNSwift – Support Information

Need Help with DNSwift?

If you're experiencing issues or have questions about using DNSwift, please review the FAQ section first. If your issue persists or isn’t listed, feel free to reach out directly.

Contact Support

  • Email:
    chatdnswift@hotmail.com
    (Typical response time: within 24–48 hours)

When Contacting Support, Please Include:

To help us assist you faster, please include the following details in your message:

  • Your Windows version (e.g., Windows 10 64-bit)
  • The version and build of DNSwift (e.g., 1.0 Build 2025.05.12)
  • A brief description of the issue
  • Screenshots or error messages (if available)

Additional Tips:

  • If DNS settings fail to apply, try temporarily disabling antivirus or firewall software and try again.
  • Restart your network adapter or computer if changes do not take effect immediately.

Stay Updated

New builds and feature updates will be announced via email or the official page (if available in the future). Stay tuned.
