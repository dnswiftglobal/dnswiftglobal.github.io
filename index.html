DNSwift 1.1 Build 25.06.12 is Released!

DNSwift is an enhanced Windows application that simplifies changing DNS server settings on your network adapters. Version 1.1 adds new features including network monitoring, DNS ping testing, and expanded DNS provider options.

New Features in v1.1:

  • Expanded DNS Provider Support: Added 7 new DNS providers:

    • NextDNS (45.90.28.0, 45.90.30.0)
    • Comodo Secure DNS (8.26.56.26, 8.20.247.20)
    • Neustar UltraDNS (156.154.70.3, 156.154.71.3)
    • CleanBrowsing DNS (185.228.168.9, 185.228.169.9)
    • AdGuard DNS (94.140.14.14, 94.140.15.15)
    • Yandex DNS (77.88.8.8, 77.88.8.1)
    • Alternate DNS (76.76.19.19, 76.223.122.150)
  • Network Monitoring: Real-time display of network traffic (bytes in/out)
  • DNS Ping Testing: Continuous ping monitoring of active DNS servers
  • Improved Widget Mode: Compact view with essential network information
  • Enhanced IP Handling: Better detection of local and public IP addresses
  • Registry Cleaner: Removes Google Chrome policies that might interfere with DNS settings

Existing Features:

  • Easy DNS switching for network adapters
  • Predefined DNS providers (Google, Cloudflare, OpenDNS, Quad9)
  • Custom DNS server support
  • Reset to default (DHCP) functionality
  • Multi-language support (English, Indonesian, Chinese, Thai, Vietnamese)
  • Persistent settings saving
  • Always-on-top option

Usage Guide

  1. Network Adapter Selection
    • Open DNSwift
    • Select your active network connection from the dropdown
  2. DNS Provider Selection
    • Choose from:
      • Default (DHCP)
      • Google DNS
      • Cloudflare DNS
      • OpenDNS
      • Quad9
      • 7 new providers
      • Custom...
  3. Custom DNS EntryÂ (Optional)
    • Enter primary and secondary DNS addresses in the format: XXX.XXX.XXX.XXX
  4. Applying Settings
    • Click "Apply" to set the selected DNS
    • Notification will confirm success or show errors
  5. Widget Mode
    • Minimizes to a compact view showing:
      • Current DNS servers
      • Network traffic statistics
      • DNS ping status
      • Local and public IP addresses
  6. Advanced Features
    • Click the network traffic labels to copy IP addresses
    • Use the switch icon to toggle always-on-top mode
    • Access the widget from system tray when minimized

Improvements from v1.0:

  • More reliable DNS switching process
  • Better error handling and notifications
  • Reduced resource usage
  • Enhanced user interface
